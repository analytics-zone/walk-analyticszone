Questions Fréquemment Posées

Comment synchroniser mes entraînements ?

L'application se synchronise automatiquement avec Apple Santé pour importer les entraînements de marche enregistrés par tout appareil ou application compatible. Assurez-vous d'avoir accordé les permissions de l'application Santé dans les Réglages iOS.

Mes données sont-elles privées ?

Oui, toutes les données sont traitées localement sur votre appareil. Nous ne collectons, ne stockons ni ne transmettons aucune de vos informations personnelles. Lire notre politique de confidentialité complète.

Comment exporter mes données ?

Vous pouvez exporter vos données d'entraînement et vos analyses dans plusieurs formats (JSON, CSV, HTML, PDF) directement depuis l'application. Tous les exports sont générés localement sur votre appareil.

Ai-je besoin d'une connexion internet ?

Non, l'application fonctionne complètement hors ligne. Tous les calculs et traitements de données se font localement sur votre appareil.

Puis-je utiliser cette application sur plusieurs appareils ?

L'application peut être installée sur tous vos appareils iOS utilisant le même identifiant Apple. Cependant, les données sont stockées localement sur chaque appareil à moins que vous n'activiez les sauvegardes d'application iOS via iCloud.

Besoin de plus d'aide ?

Vous ne trouvez pas ce que vous cherchez ? Envoyez-nous un email à analyticszone@onmedic.org et nous vous répondrons dès que possible.
