Politique de Confidentialité

Dernière mise à jour : 27 octobre 2025

Votre Vie Privée est Protégée

WalkAnalytics ne collecte, ne stocke ni ne transmet aucune de vos données personnelles. Tout le traitement des données se fait localement sur votre appareil.

Aperçu

WalkAnalytics est conçu avec votre vie privée comme priorité absolue. Cette Politique de Confidentialité explique notre engagement à protéger vos informations personnelles et vos données.

En bref : Nous ne collectons pas vos données car nous n'en avons pas besoin. Tout ce que vous faites dans l'application reste sur votre appareil.

Informations que nous ne collectons PAS

WalkAnalytics ne collecte, ne transmet ni ne stocke :

  • Informations personnelles (nom, email, numéro de téléphone, etc.)
  • Données d'activité de marche (distance, pas, vitesse, temps, etc.)
  • Données de santé (fréquence cardiaque, calories, etc.)
  • Données de localisation ou coordonnées GPS
  • Informations sur l'appareil ou identifiants
  • Statistiques d'utilisation ou analyses
  • Rapports de plantage ou diagnostics
  • Toute autre information personnelle ou sensible

Comment fonctionne l'application

Tous les calculs et traitements de données dans WalkAnalytics se font entièrement sur votre appareil :

  • Vos données de marche sont stockées uniquement dans le stockage local de votre appareil
  • Tous les calculs (pas, allure, cadence, zones, etc.) sont effectués localement
  • Aucune donnée n'est jamais envoyée à nos serveurs ou à des services tiers
  • Aucune connexion internet n'est requise pour que l'application fonctionne

Stockage des Données

Toutes les données que vous saisissez ou générez dans l'application sont stockées exclusivement sur votre appareil en utilisant les mécanismes de stockage local iOS. Ces données restent sous votre contrôle total :

  • Vos données restent sur votre appareil à tout moment
  • Nous ne pouvons pas accéder à vos données
  • Si vous supprimez l'application, toutes les données sont définitivement supprimées de votre appareil
  • Les données ne sont pas synchronisées avec iCloud ou tout autre service cloud, sauf si vous activez explicitement les sauvegardes d'application iOS

Services Tiers

WalkAnalytics n'utilise aucun service tiers, y compris :

  • Pas de services d'analyse (Google Analytics, Firebase, etc.)
  • Pas de réseaux publicitaires
  • Pas de services de rapport de plantage
  • Pas d'intégrations de réseaux sociaux
  • Pas de processeurs de paiement (l'application est gratuite)

Permissions

L'application peut demander les permissions iOS suivantes uniquement si vous choisissez d'utiliser des fonctionnalités spécifiques :

  • HealthKit/App Santé : Uniquement si vous choisissez d'importer des données d'entraînement. Toutes les données restent sur votre appareil.
  • Notifications : Uniquement si vous activez les rappels d'entraînement. Aucune donnée n'est envoyée hors de votre appareil.

Toutes les permissions sont optionnelles et vous pouvez les révoquer à tout moment via les Réglages iOS.

Confidentialité des Enfants

Comme nous ne collectons aucune donnée, WalkAnalytics est sûr pour les utilisateurs de tous âges, y compris les enfants de moins de 13 ans. Cependant, nous recommandons une surveillance parentale pour toutes les activités de fitness.

Modifications de cette Politique de Confidentialité

Nous pouvons mettre à jour cette Politique de Confidentialité de temps à autre pour refléter les changements dans l'application ou les exigences légales. Tout changement sera publié sur cette page avec une date de "Dernière mise à jour" actualisée.

Important : Nous ne changerons jamais notre engagement fondamental de ne pas collecter vos données.

Contactez-nous

Si vous avez des questions concernant cette Politique de Confidentialité ou l'application, veuillez nous contacter via notre page de contact.

Résumé

WalkAnalytics est une application axée sur la confidentialité. Nous croyons que vos données vous appartiennent et doivent rester avec vous.

  • ❌ Pas de collecte de données
  • ❌ Pas de suivi ou d'analyse
  • ❌ Pas de services tiers
  • ✅ Traitement 100% local
  • ✅ Contrôle total sur vos données
