개인정보 처리방침

최종 업데이트: 2025년 10월 27일

당신의 프라이버시는 보호됩니다

WalkAnalytics는 사용자의 개인 데이터를 수집·저장·전송하지 않습니다. 모든 데이터 처리는 전적으로 사용자의 기기 내에서 이루어집니다.

개요

WalkAnalytics는 설계 단계부터 프라이버시를 최우선 가치로 두고 만들어졌습니다. 이 개인정보 처리방침은 우리가 어떻게 사용자의 정보를 보호하는지 설명합니다.

요약하면: 우리는 사용자의 데이터를 필요로 하지 않기 때문에 수집하지 않습니다. 앱에서 이루어지는 모든 활동은 사용자의 기기 안에만 머뭅니다.

우리가 수집하지 않는 정보

WalkAnalytics는 다음과 같은 정보를 수집·전송·저장하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 걷기 활동 데이터(거리, 걸음 수, 속도, 시간 등)
  • 심박수, 칼로리 등 건강 데이터
  • 위치 정보 또는 GPS 좌표
  • 기기 정보 또는 식별자
  • 사용 통계나 분석 데이터
  • 크래시 리포트나 진단 정보
  • 그 외 기타 개인·민감 정보

앱 동작 방식

WalkAnalytics의 모든 계산과 데이터 처리는 전적으로 사용자의 기기에서 이루어집니다.

  • 걷기 데이터는 iOS 로컬 저장소에만 보관됩니다.
  • 걸음 수, 속도, 케이던스, 존 계산 등은 모두 기기에서 수행됩니다.
  • 데이터는 당사 서버나 제3자 서비스로 전송되지 않습니다.
  • 앱을 사용하는 데 인터넷 연결이 필요하지 않습니다.

데이터 저장

앱에서 생성되는 모든 데이터는 iOS의 로컬 저장 메커니즘을 사용하여 기기 내에만 저장됩니다.

  • 데이터는 항상 사용자의 기기에만 남습니다.
  • 기기를 교체하거나 앱을 삭제하면 데이터도 함께 삭제될 수 있습니다.
  • iCloud 백업을 활성화한 경우, 백업·복원 작업은 Apple 인프라에서 처리됩니다.

제3자 서비스

WalkAnalytics는 광고 네트워크, 분석 SDK, 소셜 로그인, 외부 추적 스크립트를 사용하지 않습니다.

  • Google Analytics, Firebase Analytics 등 분석 도구 미사용
  • Facebook, Google 등 소셜 로그인·추적 코드 미사용
  • 광고 SDK나 푸시 알림용 마케팅 SDK 미사용
