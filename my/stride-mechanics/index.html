လျှော့လုံးကျ အလျှောက်လှမ်း ယန္တရား

လူမှုမီဒီယာ အလျှောက်လှမ်း အခြင်းအရာ ခွဲခြမ်းစိတ်ဖြာမှု

အလျှောက်လှမ်းခြင်းသည် အင်္ဂါရုပ်ကွန်ယက်များ အများအပြားက ဆက်စပ်နေသော ရုပ်ခွင်းဆိုင်ရာ ကြွေးမုံးမှုတစ်ခုဖြစ်သည်။ အလျှောက်လှမ်း ယန္တရားကို နားလည်ခြင်းဖြင့် အရည်အသွေး မြှင့်တင်ခြင်း၊ ဒဏ်ရာကင်းဆိုင်ရာ ကာကွယ်မှု၊ နှင့် စွမ်းဆောင်ရည် တိုးတက်မှုများကို အများအားဖြင့် ပြုလုပ်နိုင်သည်။ ဤ လမ်းညွှန်ချက်သည် သာမန် အလျှောက်လှမ်းမှ အလျှော့ လွှမ်းခြင်း အခွင့်အလမ်းကို တိုင်းတာပြီး သည်လည်းတစ်ခါတလည် သုံးသပ်မှုများ ပေးစွမ်းသည်။

အလျှောက်လှမ်း ယန္တရား ကွင်း

အလျှောက်လှမ်း ယန္တရား ကွင်း တစ်ခုမှာ တူညီသော ခြင်း၏ ဆက်စပ်သော ခြင်းအချင်းချင်း၏ အခိုက်အတန်း၏ အချိန်ကြေး သည်။ အလျှောက်လှမ်းခြင်းတွင် သမိုဒ္ဒ မြေပြင်ဆက်စပ်မှု ရှိပြီး ခြင်းနှစ်ခုက သမိုဒ္ဒ မြေပြင်ပေါ်တွင် ရှိသည့် ထူးခြားသော နှစ်ခန္ဒ ကျင့်သုံးမှု ကွင်းရှိသည်။

ကွင်း ကွင်း၏ % အဓိက ကိုင်တွယ်မှုများ
ခြင်း ပုံသေလုပ်ငန်း 60% ခြင်း မြေပြင်ဆက်စပ်
စွဲမုံး ကွင်း 40% ခြင်း လေထဲ၊ ရှေ့သို့ လှုံ့ဆော်
နှစ်ခန္ဒ ကျင့်သုံးမှု 20% ခြင်းနှစ်ခု မြေပြင်ပေါ်တွင် (အလျှောက်လှမ်းခြင်းကို အဓိကကျ)

ခြင်း ပုံသေလုပ်ငန်း ခွဲခြမ်းစိတ်ဖြာမှု (ကွင်း ၆၀%)

မြေပြင်ဆက်စပ်အတွင်း ထူးခြားသော ခွဲခြမ်းကွင်း ငါးခုရှိ:

  1. ကနဦး ဆက်စပ်မှု (ခြင်း အချိုး):
    • ခြင်း မြေပြင်ဆက်စပ် ~10° dorsiflexion မှ
    • ဒူးချောင်း အတန်းတော် (~180-175°)
    • အလယ် အလွယ် ~30°
    • ပထမ အလိုင်း အားအလုံးလုံး စတင် (~110% ခန္ဒ အလုံးလုံး)
  2. ဘောင်းဖိုးခွင့်ခြင်း (ခြင်း အပြည့်အစုံ):
    • အပြည့်အစုံ ခြင်း ဆက်စပ် 50ms အတွင်း စည်းခြင်း
    • ခွင့်ခြင်း ကူးပြောင်း ခြင်း အလယ် သို့မဟုတ် အလယ်တွင်
    • ဒူးချောင်း အလွယ် 15-20° အကျ ယန္တရား ဖြည်းဆည်းခြင်း
    • ခြင်း အလွယ် plantarflexes အတွက် အပြည့်အစုံ ခြင်း အနေအထား
  3. အလယ်ခွင့်ခြင်း:
    • ခန္ဒ အလုံးလုံး အလယ်ဖန်ဆိုင် သည့် ခြင်းအသုံးပြုမှုတွင် ကျေးဇူးပြု
    • ဆန့်ကျင်ဖက် ခြင်း စွဲမုံးခြင်း
    • ခြင်း dorsiflexes အခြင်း အလုံးလုံး အစီအမံ
    • အနိမ့ဆုံး အလိုင်း အားအလုံးလုံး (80-90% ခန္ဒ အလုံးလုံး)
  4. အဆုံး ခြင်း ပုံသေလုပ်ငန်း (ခြင်း မြင့်ခြင်း):
    • ခြင်း မြေပြင်မှ မြင့်ခြင်းစတင်
    • ခွင့်ခြင်း အလယ် နှင့် ခြင်းခုံများ သို့မဟုတ်အဖြုံး ခွင့်ခြင်း
    • ခြင်း plantarflexion စတင်
    • အလယ် တိုးချဲ့မှု အများဆုံး (~10-15°)
  5. ကိုယ်စိတ်အလွယ် (ခြင်းခုံ-မှ):
    • အဆုံး ခွင့်ခြင်း အလယ် ခွင့်ခြင်း
    • ဒုတိယ အလိုင်း အားအလုံးလုံး (~110-120% ခန္ဒ အလုံးလုံး)
    • လျင်မြန်သော ခြင်း plantarflexion (အထိ 20°)
    • ဆက်စပ် အချိန်: 200-300ms စုစုပေါင်း

စွဲမုံး ကွင်း ခွဲခြမ်းစိတ်ဖြာမှု (ကွင်း ၄၀%)

ခြင်းကို ရှေ့သို့ လှုံ့ဆော်စေသည့် ခွဲခြမ်းကွင်း သုံးခု:

  1. ကနဦး စွဲမုံး:
    • ခြင်းခုံ မြေပြင်မှ ထွက်
    • ဒူးချောင်း လျင်မြန်စွာ အလွယ် ~60° (အများဆုံး အလွယ်)
    • အလယ် ဆက်လက် အလွယ်
    • ခြင်း ရှင်းလင်း မြေပြင် 1-2cm
  2. အလယ်-စွဲမုံး:
    • စွဲမုံး ခြင်း ကျေးဇူးပြု ခြင်း ကျေးဇူးပြု
    • ဒူးချောင်း တိုးချဲ့မှု စတင်
    • ခြင်း dorsiflexes ကျီးမှုတျ့တျုံ့မှု
    • အနည်းဆုံး မြေပြင် ရှင်းလင်း
  3. အဆုံး စွဲမုံး:
    • ခြင်း တိုးချဲ့မှု ခြင်း အချိုးလုံးခြင်း အတွက် ပြင်ဆင်
    • ဒူးချောင်း အပြည့်အစုံ တိုးချဲ့မှု အဆင့်
    • အရီခြင်း ခြင်း ခေါ်ယူရန် စတင်
    • ခြင်း သည့် အလွယ် dorsiflexion အတွင်း ထိန်းသိမ်း

မရှိမဖြစ်လိုအပ်သော ယန္တရား ပာမီတာများ

အလျှောက်လှမ်း အလျှည အလျင်မြန် အလျဉ್ဉ ခွဲခြမ်းစိတ်ဖြာမှု

အရေးကြီးသော ခွဲခြမ်းစိတ်ဖြာမှု:

  • အလျှည အလျင်မြန်: ခြင်းတစ်ခု၏ ခြင်းမှ ဆန့်ကျင်ဖက်ခြင်း ခြင်း အထိ အကွာအဝေး (လက်ဝဲ→လက်ယာ သို့မဟုတ် လက်ယာ→လက်ဝဲ)
  • အလျှောက်လှမ်း အလျည: ခြင်းတစ်ခု၏ ခြင်းမှ တူညီသော ခြင်း ခြင်း အချိုး အထိ အကွာအဝေး (လက်ဝဲ→လက်ဝဲ သို့မဟုတ် လက်ယာ→လက်ယာ)
  • ဆက်ဆံမှု: အလျှောက်လှမ်း အလျည = အလျှည အလျင်မြန် နှစ်ခု
  • သဋ္ဋုတ်မှု: ကျန်းမာသော အလျှောက်လှမ်းခြင်းတွင় လက်ယာ နှင့် လက်ဝဲ အလျှည အလျင်မြန်များသည် အချင်းချင်း ၂-၃% အတွင်း ရှိသည်
အလျား (စင်တီမီတာ) အကောင်းဆုံး အလျှောက်လှမ်း အလျည (မီတာ) အလျား၏ %
150 0.60-0.75 40-50%
160 0.64-0.80 40-50%
170 0.68-0.85 40-50%
180 0.72-0.90 40-50%
190 0.76-0.95 40-50%

အလယ်တန်း အလျှောက်လှမ်း သမားများ အလျား၏ 70% အထိ အလျှောက်လှမ်း အလျည ရယူသည့်အရာ အားဖြင့် မြတ်သောအခွင့်အလမ်း နှင့် အလယ်စွမ်းရည် ယေဘုယျ ရယူသည်။

အလျှည အလျင်မြန် အကောင်းဆုံးအားဖြင့်

တစ်မိနစ်လျှင် အလျှည (spm) များသည် ယန္တရား၊ အရည်အသွေး၊ နှင့် ဒဏ်ရာ အခန်းကဏ္ဍကို အလွန်ကျယ်လွန်းသည့်အားဖြင့် တည်ထောင်သည်:

အလျှည အလျင်မြန် အဆင့် အကဲအဖြတ် ယန္တရား ထူးခြားချက်များ
<90 spm အလွန်ကျ အလျန်လျှည၊ အလွန်ကျ တည်းဖြတ် အားအလုံးလုံး၊ အနိမ့ဆုံး အရည်အသွေး
90-99 spm ကျ အလယ်အလတ်တန်း အဆင့် အောက်ခြင်း
100-110 spm အလယ်အလတ်တန်း ကိုယ်စည်းကွယ်လွန်ထားသော အလျည/အလျွန်၊ 3-4 METs
110-120 spm လျင်မြန်သော အလယ်အလတ်တန်း-အလွန်ကျ၊ အကောင်းဆုံး ကျန်းမာရေး
120-130 spm အလွန်ကျ စွမ်းအားလျှောက်လှမ်း၊ 5-6 METs
130-160 spm အလျှောက်လှမ်း သမား ခွင့်အဆင့် အခွင့်အလမ်း အပြည့်အစုံ
သုတေသန ရှာဖွေခွင့်: CADENCE-Adults ကျေးဇူးပြုမှု (Tudor-Locke et al., 2019) သည် 100 spm သည် အလယ်အလတ်တန်း အဆင့် (3 METs) သည့် အင်္ဂါရုပ် ၈၆% အခွင့်အလမ်း နှင့် ၈၉.၆% အခွင့်အလမ်း အလယ်တန်း အထူး ၂၁-၈၅ အထူး လူများအတွက် အဖြေများ ဖြောင့်စောင့်မည်။

မြေပြင် ဆက်စပ် အချိန်

စုစုပေါင်း ခြင်း ပုံသေလုပ်ငန်း ကြေးအချိန်: 200-300 milliseconds

  • သာမန် အလျှောက်လှမ်း (4 km/h): ~300ms ဆက်စပ် အချိန်
  • လျင်မြန်သော အလျှောက်လှမ်း (6 km/h): ~230ms ဆက်စပ် အချိန်
  • အလွန်လျင်မြန်သော အလျှောက်လှမ်း (7+ km/h): ~200ms ဆက်စပ် အချိန်
  • အလျှည အလျင်မြန်အတွင်း 比較: အလျှည အလျင်မြန် <200ms ဆက်စပ်၊ ရုပ်ခွင်းလုံးလုံးအားဖြင့် အဖြစ်

ဆက်စပ် အချိန် အလျင်မြန် တိုးလျင်သည့်အခါ ကျဆင်း အကြောင်းအရာများ:

  1. ခြင်း ပုံသေလုပ်ငန်း အချိန် ကွင်း အတွင်း အတိုးအလျှော့
  2. ခွင့်ခြင်း ကူးပြောင်း ပိုမိုလျင်မြန်
  3. ကျေးဇူးပြုခြင်း ကိုယ်မ တင်ပြမီ မြင့်မြင့် အင်္ဂါရုပ်
  4. ပိုမိုကြီးမားသော elastic စွမ်းအား သိုလှောင်ခြင်း နှင့် ပြန်လည်အသုံးပြုခြင်း

နှစ်ခန္ဒ ကျင့်သုံးမှု အချိန်

ခြင်းနှစ်ခု သမိုဒ္ဒ မြေပြင်ပေါ်တွင် ရှိသည့် အချိန်သည် အလျှောက်လှမ်းခြင်းကို အဓိကကျ ဖြစ်ပြီး အလျှည အလျင်မြန်တွင် အပျောက်ခွင့်ရှိ (ရုပ်ခွင်းလုံးလုံးအားဖြင့် အဖြစ်ခွင့် အဖြေထုတ်ခြင်း)။

နှစ်ခန္ဒ ကျင့်သုံးမှု % အကဲအဖြတ် ဆေးခန်း အရေးပါမှု
15-20% သာမန် (လျင်မြန်သော အလျှောက်လှမ်း) ကျန်းမာသော၊ အားထည့်သွင်းခွင့် အလျှောက်လှမ်း
20-30% သာမန် (အလယ်အလတ်တန်း အလျှောက်လှမ်း) အများအားဖြင့် အမြဲတမ်း အလျှည အလျွန်များအတွက်
30-35% အပြည့်အစုံ အလျှောက်လှမ်း အချိုးအဝန်း ကိုင်တွယ်မှု အလွန်စုံမွေ့မှုများ ဖြစ်ရှိနိုင်
>35% မြင့်မြင့်သော ကျံုးကျ ဒဏ်ရာ အခန်းကဏ္ဍ ဆေးခန်း အစီအစဉ် အကြုံးအကျ습်

Apple HealthKit ပေါင်းစည်းခြင်း: iOS 15+ သည် နှစ်ခန္ဒ ကျင့်သုံးမှု ရာခိုင်နှုန်းကို ရုပ်ခွင်း အဆင့် အလည်အင်္ဂါရုပ်အဖြစ် တိုင်းတာသည့်အားဖြင့် တန်ဖိုးများသည် >35% အခွင့်အလမ်း အဖြေထုတ်ခြင်း "အနိမ့ဆုံး" အလျှောက်လှမ်း ယုံကြည်ချက်အဖြစ် အလုံးအစုံကျရွေစည်စဉ်၊

အလုံးလုံး အလေ့အကျ

ခန္ဒ အလုံးလုံး၏ အလယ်ဖန်ဆိုင် အထက်သို့လည်း အပ်တွင် အခွင့်အလမ်း အဝေးအတွင်း ရွေ့လျားမှု:

  • သာမန် အဆင့်: 4-8 စင်တီမီတာ
  • အကောင်းဆုံး အရည်အသွေး: ~5-6 စင်တီမီတာ
  • အလွန်ကျ (>8-10 စင်တီမီတာ): အသုံးအဆုံး မလိုအပ်သော အလုံးလုံး အလေ့အကျမှ စွမ်းအား စုံမွေ့မှု
  • အလွန်နည်း (<4 စင်တီမီတာ): ရုပ်ခွင်းလုံးလုံးအားဖြင့် အလျှောက်လှမ်း၊ ဖြစ်ရှိနိုင်သော ကျေးဇူးပြုခြင်း

အလုံးလုံး အလေ့အကျ သည့် ယန္တရားများ:

  1. အလယ်စွမ်းရည် အလုံးလုံးအားဖြင့် transverse အင်္ဂါရုပ်တွင် အလုံးလုံးဆုံ (4-8°)
  2. အလယ်စွမ်းရည် အလုံးလုံးအားဖြင့် frontal အင်္ဂါရုပ်တွင် tilt (5-7°)
  3. ဒူးချောင်း အလွယ် ခြင်း ပုံသေလုပ်ငန်း အတွင်း (15-20°)
  4. ခြင်း plantarflexion-dorsiflexion coordination
  5. အလိုက်အလျှား အလယ်စွမ်းရည် အလုံးလုံး shift (~2-5 စင်တီမီတာ)

မြင့်မြင့်သော ယန္တရား အကွင်းများ

လက်သည် စွဲမုံး ယန္တရား

ကိုယ်စည်းကွယ်လွန် လက်သည် အလုံးအစုံလုံးခြင်း အလှုံ့အဆောင်အလုံးအစုံမဟုတ်—ဒါဟ အရေးကြီးသော ယန္တရား အကျိုးအကျန်များ ပေးစွမ်းသည်:

စွမ်းအား သိုလှောင်ခြင်း: သမိုဒ္ဒ လက်သည် စွဲမုံးခြင်း သည် အခုလုံးလုံးအားဖြင့် အလုံးလုံးခြင်းအတွင်း အသုံးအဆုံး အမြတ် 10-12% ကြည့်ရှု့ (Collins et al., 2009)။

အကောင်းဆုံး လက်သည် စွဲမုံး ထူးခြားချက်များ:

  • အလုံးအစုံလုံးခြင်း: Contralateral ကိုယ်စည်းကွယ်လွန် (လက်ဝဲ လုံးအစုံ ခြင်း ယာ)
  • အဆင့်: 15-20° anterior-posterior အလုံးလုံးအဖြစ် အလုံးလုံးအားဖြင့် ထွက်လွင်
  • အက္ခရာချုပ် အလုံးအစုံ: 90° အလွယ် စွမ်းအားလျှောက်လှမ်းအတွက်; 110-120° အကျိုးအကျန် အလျှောက်လှမ်းအတွက်
  • လက်သည် အနေအထား: အလုံးလုံးလုံးခြင်း၊ ခန္ဒ အလုံးလုံးအားဖြင့် အခွင့်အလမ်းမှ မထုံအရင်မ
  • အခေါ်အဝေါ် အလုံးလုံးဆုံ: အလွန်နည်း အလုံးလုံးဆုံ၊ လက်သည်များ အစီအစဉ်ခြင်း မြေခွဲမှ စွဲမုံးခြင်း

ယန္တရား လုပ်ဆောင်ချက်များ:

  1. အလုံးလုံးဆုံ လုံးလုံးအားဖြင့် အဖြေထုတ်ခြင်း: လက်သည်များ ခြင်း အလုံးလုံးဆုံ သည့် သည့် သည့်ကွင်း မြေခွဲ့စ
  2. အလုံးလုံးအားဖြင့် မြေပြင် အကျိုးအကျန်များ အလုံးလုံးအားဖြင့် အတိုးအလျှော့: အလုံးလုံးအားအလုံးလုံး အလုံးအစုံကျ သည့် အမြတ်အုပ်
  3. ကိုယ်စည်းကွယ်လွန် တိုးတက်မှု: အလုံးလုံးဆုံ့ိုင်ရာ၊ အခုံ အလုံးလုံးအဖြစ် အတိုးအလျှော့
  4. စွမ်းအား ကူးပြောင်း: ခွင့်ခြင်းအလုံးခြင်း အတွင်းအဖြစ် အကူအညီ လုပ်ဆောင်

ခြင်း အချိုး အမျိုးအစားများ

ခြင်းလျင်မြန်သော ၈၀% သည် ကိုယ်စည်းကွယ်လွန် ခြင်း-အချိုး အလုံးအစုံ (ခြင်း-အလုံးအစုံ အချိုး) အလုံးလုံးခြင်း။ အခြား အလုံးအစုံများ ရှိသည်သူ် အများများက အလွန်နည်း:

အချိုး အလုံးအစုံ အများအုံ့အုံ့ရွေ့ ထူးခြားချက်များ
ခြင်း အချိုး ~80% ခြင်း အချိုး့ခန့် ~10° dorsiflexion၊ M-shaped အားအလုံးလုံး အကြွေးလုံးခြင်း
အလယ်ခြင်း အချိုး ~15% ခြင်း အပြည့်အစုံ ပြင်ဆင်ခြင်း၊ အလွန်ကျ တည်းဖြတ် အလုံးအစုံ အလုံးအစုံ၊ အတိုးအလျှော့ အလျည
အလုံးအစုံ ခြင်း အချိုး ~5% အလျှောက်လှမ်းခြင်းတွင် အလွန်နည်း၊ အလွန်လျင်မြန်သော အလျှောက်လှမ်း သမား အတုတင်ခြင်း အတွင်း ရွေ့

ခြင်း အချိုး အားအလုံးလုံးအားဖြင့် မြေပြင்:

  • ပထမ အလုံးအစုံ (~50ms): တည်းဖြတ် အလုံးအစုံ အတုတင်ခြင်း၊ 110% ခန္ဒ အလုံးလုံး
  • အနည်းဆုံး (~200ms): အလယ်ခွင့်ခြင်း အခွင့်အလမ်း၊ 80-90% ခန္ဒ အလုံးလုံး
  • ဒုတိယ အလုံးအစုံ (~400ms): ခွင့်ခြင်း ခွင့်ခြင်း၊ 110-120% ခန္ဒ အလုံးလုံး
  • စုစုပေါင်း အားအလုံးလုံးအားဖြင့် အကြွေးလုံးခြင်း: ထူးခြားသော "M" သို့မဟုတ် နှစ်ခုခွန်းမျက်နှာစုံ အလုံးအစုံ

အလယ်စွမ်းရည် နှင့် အလယ် ယန္တရား

အလယ်စွမ်းရည် အလုံးလုံးအားဖြင့် အလုံးလုံးဆုံ ကုန်သုံးခွင်း ထွက်လွင် အလုံးလုံးအဖြစ်၊ ကျေးဇူးပြုခြင်းအလုံးလုံးအားဖြင့် အခွင့်အလမ်း အားသုံး အဖြစ်သည့် အလုံးလုံး:

1. အလယ်စွမ်းရည် အလုံးလုံးဆုံ (Transverse Plane):

  • သာမန် အလျှောက်လှမ်း: 4-8° အလုံးလုံးဆုံ တစ်ဖက်စီ
  • အလျှောက်လှမ်း သမား: 8-15° အလုံးလုံးဆုံ (ကျေးဇူးပြုခြင်း အလျည အလျွန်များအတွက်)
  • လုပ်ဆောင်ချက်: လုံးလုံးအားဖြင့် အလျည ခြင်းတိုးချဲ့ခြင်း၊ အလျှောက်လှမ်း အလျည တိုးချဲ့မှု
  • ကိုယ်စည်းကွယ်လွန်: အလယ်စွမ်းရည် အလုံးလုံးဆုံ လုံးဆုံ ခြင်း ရုပ်ခွင်း အလုံးလုံးအားဖြင့် ကျေးဇူးပြုခြင်း

2. အလယ်စွမ်းရည် အလုံးလုံးအားဖြင့် Tilt (Frontal Plane):

  • အဆင့်: 5-7° drop စွဲမုံးခြင်း-အခြေ အလယ် သည့်အံုံ
  • Trendelenburg အလျှောက်လှမ်း: အလွန်ကျ drop အလယ် abductor အားနည်းမှု အလုံးအစုံ
  • လုပ်ဆောင်ချက်: ခန္ဒ အလုံးလုံး အလယ်အလတ်တန်း အလေ့အကျ သည့်အားဖြင့် အလုံးလုံးအလုံးခြင်း သည့် အလုံးလုံးအားဖြင့် အလုံးအစုံ ကျဆင်း

3. အလယ်စွမ်းရည် အလုံးလုံး Shift (Frontal Plane):

  • အလိုက်အလျှား အလုံးအစုံ: 2-5 စင်တီမီတာ အကျင့် ခြင်းအပေါ် အလုံးအစုံ သည့်အားဖြင့်
  • လုပ်ဆောင်ချက်: အချိုးအဝန်း အလုံးအစုံ ထိန်းသိမ်း၊ ခန္ဒ အလုံးလုံးအားဖြင့် အံုံအရင်မြစ်

အလုံးလုံး အလုံးအနေအထား နှင့် အစီအမံ

အကောင်းဆုံး အလျှောက်လှမ်း အလုံးအနေအထား:

  • အလုံးလုံး အနေအထား: အလုံးလုံးအလုံးအနေ သို့မဟုတ် 2-5° ရှေ့အလုံးအစုံ ankle မှ
  • ခေါင်း အစီအမံ: ကျီးမှုတျ့တျုံ့မှု၊귀များ သည့် အခေါ်အဝေါ်များ
  • အခေါ်အဝေါ် အနေအထား: အလုံးလုံးလုံးခြင်း၊ မြင့်မြင့်သော အများအားဖြင့်ခွင့်
  • အလုံးလုံး အကျင့်၊ အလုံးအစုံ: အလယ်အလတ်တန်း အကျင့်အလုံးအစုံ အလုံးလုံးအားဖြင့် တည်ဆောက်ခြင်း
  • ကြည့်ရှုခြင်း အလုံးအစုံ: 10-20 အမီတာ အကျယ်အဝန်း အပြည့်အစုံ ခြင်း အအီတီ

အများအားဖြင့် အလုံးအနေအထား အမှားအကျးတွေ:

  • အလွန်ကျ ရှေ့အလုံးအစုံ: အများအားဖြင့် အားနည်း အလယ် extensors မှ
  • နောက်အလုံးအစုံ: အလျင်မြန်သည့် အခါ၊ အဆီအမျိုးမျိုး၊ သို့မဟုတ် အားနည်း abdominals
  • အလိုက်အလျှား အလုံးအစုံ: အလယ် abductor အားနည်းမှု သို့မဟုတ် ခြင်း အလျား အခြားအံုံ
  • ခေါင်း ရှေ့အလုံးအစုံ: အုပ်ခြုံချုပ် အလုံးအနေအထား၊ အချိုးအဝန်း အလုံးအစုံ သည့် အံုံ

အလျှောက်လှမ်း သမား အခွင့်အလမ်း

အလျှောက်လှမ်း သမား သည် အခွင့်အလမ်း ယန္တရား ခွင့် (World Athletics ခွင့် 54.2) သည့်အားဖြင့် အကျည်းအုံ့ခွင့် သည့် အလုံးအစုံ အလျွန်အလုံးအစုံ အလုံးလုံးအားဖြင့် အအံုံ အလုံးအစုံ မြင့်မြင့်သော အလျွန် မြင့်မြင့်သော အလုံးအစုံ။

နှစ်ခုမြုံ့ အခုံးမြုံ့ ခွင့်များ

ခွင့် 1: အလုံးလုံးအားဖြင့် ဆက်စပ်မှု

  • မြေပြင်အားဖြင့် ရွေ့လျားမှု သည့် အံုံ အရင်မြစ် (အလုံးအစုံ အဖြစ် အည)
  • ရှေ့ငယ်ခြင်း အုံံ မြေပြင် ဆက်စပ်ခြင်း အကျင့် နောက်အုံံ ခြင်း မြေပြင်မှ ထွက်
  • အုံံ အလုံးအစုံ ဆွဲခြင်း အုံံ အုံံ ၅၀မီတာ အုံံ ခွင့်များ
  • အကျည်းအုံ့ အလျှောက်လှမ်း သမားများ အလုံးလုံးအားဖြင့် အံုံ အအုံံ 13-15 အလျွန်/အုံံ အုံံ အလုံးအစုံ ဆက်စပ်မှု

ခွင့် 2: အစီအမံ ခြင်း အလုံးအလျင်မြန်

  • အများအားဖြင့် အုံံ ခြင်း အစီအမံ (ခွင့် အလုံးလုံးခြင်း မဟုတ်) ကနဦး ဆက်စပ်မှု အကျင့် အလုံးလုံးအားဖြင့် အလုံးလုံး အလုံးအနေအထား
  • ဒူးချောင်း အလုံးလုံးခြင်း အကျင်းအုံံ ကနဦး ဆက်စပ်မှု အလယ် အုံံ အုံံ မြေခွဲမဟုတ်ခြင်း
  • အုံံ အုံံ၊ အုံံ အုံံ 3-5° အုံံ အုံံ အုံံ အုံံ မြေခွဲမဟုတ် အုံံ
  • ဤ ခွင့် အုံံ အလျှောက်လှမ်း သမား သည့်အားဖြင့် အုံံ သို့မဟုတ် စွမ်းအားလျှောက်လှမ်းခြင်း အုံံ

အလျွန် အုံံ အုံံ အုံံ အုံံ အုံံ

ခွင့် 130-160 spm အုံံ အုံံ အုံံ အုံံ အုံံ:

  1. အုံံ အလုံးအစုံ အုံံ အုံံ:
    • 8-15° အုံံ အုံံ (vs. 4-8° သာမန် အလျှောက်လှမ်း)
    • အုံံ အုံံ ခြင်း အုံံ တိုးချဲ့
    • အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ
  2. အုံံ အုံံ အုံံ:
    • 15-20° အုံံ တိုးချဲ့ (vs. 10-15° သာမန်)
    • အုံံ အုံံ အုံံ glutes နှင့် hamstrings
    • အုံံ အုံံ အုံံ ခြင်း နောက်ဖက်မှ တိုးချဲ့
  3. အုံံ လက်သည် အုံံ:
    • အုံံ အုံံ 90° (အုံံ အုံံ = အုံံ အုံံ)
    • အုံံ နောက်ဖက် အုံံ ခွင့်ခြင်း အုံံ
    • ကိုယ်စည်းကွယ်လွန် 1:1 အုံံ အုံံ အုံံ
    • လက်များ အုံံ အုံံ အုံံ အုံံ ရှိ ရှိနိုင်
  4. တိုးတက်သော မြေပြင် အုံံ အုံံ:
    • အုံံ အုံံ 130-150% ခန္ဒ အုံံ
    • အုံံ လုံးလုံးအားဖြင့် သည့် အုံံ
    • အုံံ အုံံ လက်ခြင်း နှင့် အုံံ အုံံ ပြင်ဆင်
  5. အုံံ အုံံ အုံံ:
    • အုံံ အလျှောက်လှမ်း သမား: 3-5 စင်တီမီတာ (vs. 5-6 စင်တီမီတာ သာမန်)
    • အုံံ ရှေ့ အုံံ အုံံ
    • အုံံ အုံံ အုံံ အုံံ နှင့် အုံံ အုံံ အုံံ အုံံ

အုံံ အုံံ

အလျှောက်လှမ်း သမား 13 အလျွန်/အုံံ အုံံ:

  • VO₂: ~40-50 mL/အုံံ/မိနစ် (အုံံ အုံံ 9-10 အလျွန်/အုံံ အုံံ)
  • METs: 10-12 METs (အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ)
  • အုံံ အုံံ: ~1.2-1.5 kcal/အုံံ/အုံံ (အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ)
  • Lactate: ရယူ ရှိ ရှိနိုင် 4-8 mmol/L အုံံ အုံံ

အလျှောက်လှမ်း vs အုံံ: အုံံ အုံံ

အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ သည့် အုံံ:

ပာမီတာ အလျှောက်လှမ်း အုံံ
မြေပြင် ဆက်စပ်မှု အုံံ အုံံ နှစ်ခန္ဒ အုံံ အုံံ အုံံ ရုပ်ခွင်း အုံံ
အုံံ အုံံ ~62% ကွင်း (~300ms 4 အလျွန်/အုံံ အုံံ) ~31% ကွင်း (~150-200ms)
နှစ်ခန္ဒ အုံံ 20% ကွင်း 0% (ရုပ်ခွင်း အုံံ အုံံ)
အုံံ အလုံးအစုံ အုံံ 110-120% ခန္ဒ အုံံ 200-300% ခန္ဒ အုံံ
အုံံ အုံံ အုံံ အုံံ (အုံံ↔အုံံ) Spring-mass အုံံ (elastic အုံံ)
ဒူးချောင်း အုံံ ဆက်စပ်မှု အုံံ အုံံ (~5-10°) အုံံ (~20-30°)
ခန္ဒ အုံံ အုံံ အုံံ အုံံ arc, အုံံ အုံံ အုံံ ကြီးမားသော အုံံ အုံံ
အုံံ အုံံ အုံံ အုံံ အုံံ ~7-8 အလျွန်/အုံံ အုံံ အုံံ အုံံ ~8 အလျွန်/အုံံ အုံံ

အလျှောက်လှမ်း-အုံံ အုံံ သည ~7-8 အလျွန်/အုံံ (2.0-2.2 မီတာ/စက) အုံံ ကြီးမားသည့်အကြောင်းအရာ:

  1. အလျှောက်လှမ်း သည အုံံ အုံံ အုံံ သည့် အုံံ အုံံ
  2. အုံံ အုံံ အုံံ အုံံ ဆက်စပ်မှု အုံံ
  3. အုံံ elastic အုံံ အုံံ အုံံ အုံံ
  4. အုံံ အုံံ အုံံ အုံံ အုံံ သည့် အုံံ
သုတေသန ရှာဖွေခွင့်: အုံံ အုံံ အုံံ 7 အလျွန်/အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ သည့်အားဖြင့် အုံံ အုံံ အုံံ မှ သည့် သည့် အုံံ (Margaria et al., 1963) အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ။

အုံံ အုံံ အုံံ နှင့် အုံံ

1. အုံံ အုံံ

အုံံ: ခြင်း အုံံ ခန္ဒ အုံံ အုံံ အုံံ အုံံ အုံံ

ယန္တရား အုံံ:

  • အုံံ အုံံ 20-30% ခန္ဒ အုံံ
  • တိုးတက်သော အုံံ အုံံ (130-150% vs. 110% သာမန်)
  • အုံံ အုံံ အုံံ အုံံ အုံံ
  • အုံံ ခွင့်ခြင်း အုံံ
  • တိုးတက်သော ဒဏ်ရာ အုံံ (အုံံ အုံံ၊ အုံံ အုံံ)

အုံံ:

  • အုံံ အုံံ တိုးချဲ့: လက်ရှိ spm သို 5-10% အုံံ
  • အုံံ "အုံံ အုံံ": ခြင်း အုံံ ခန္ဒ အုံံ အုံံ အုံံ
  • အုံံ အုံံ: အုံံ၊ အုံံ အုံံ
  • ရှေ့ အုံံ: အုံံ 2-3° အုံံ အုံံ အုံံ

2. အုံံ အုံံ

အုံံ: အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ

အုံံ Gait အုံံ အုံံ (GSI) အုံံ:

GSI (%) = |အုံံ - အုံံ| / [0.5 × (အုံံ + အုံံ)] × 100

အုံံ:

  • <3%: သာမန်၊ အုံံ အုံံ အုံံ
  • 3-5%: အုံံ အုံံ၊ အုံံ အုံံ အုံံ
  • 5-10%: အုံံ အုံံ၊ အုံံ အုံံ အုံံ
  • >10%: အုံံ အုံံ၊ ပညာရှင် အုံံ အုံံ

အုံံ အုံံ:

  • အုံံ ဒဏ်ရာ သို့မဟုတ် အုံံ (အုံံ ခြင်း အုံံ)
  • ခြင်း အုံံ အုံံ (>1 စင်တီမီတာ)
  • အုံံ အားနည်း (အုံံ abductors, glutes)
  • ကြီးမန်းအုံံ အုံံ (အုံံ၊ အုံံ)
  • အုံံ အုံံ အုံံ

အုံံ:

  • စွမ်းအား သင်ခန်း: အုံံ-ခြင်း အုံံ အုံံ ခြင်း အုံံ
  • အုံံ အုံံ: အုံံ-ခြင်း အုံံ၊ အုံံ အုံံ
  • Gait အုံံ: Metronome-အုံံ အုံံ၊ အုံံ အုံံ
  • ပညာရှင် အုံံ: ရုပ်ခွင်းအုံံ၊ အုံံ၊ အုံံ

3. အုံံ အုံံ အုံံ

အုံံ: ခန္ဒ အုံံ 8-10 စင်တီမီတာ အုံံ ရွေ့လျားမှု

ယန္တရား အုံံ:

  • အုံံ အုံံ (ရှေ့ ခွင့်ခြင်း အုံံ)
  • အုံံ အုံံ 15-20% တိုးတက်မှု
  • အုံံ အုံံ အုံံ အုံံ
  • အုံံ အုံံ အုံံ အုံံ အုံံ

အုံံ:

  • အုံံ "အုံံ ရှေ့": အုံံ အုံံ အုံံ အုံံ
  • အုံံ အုံံ: အုံံ၊ အုံံ အုံံ
  • အုံံ အုံံ: အုံံ အုံံ နှင့် tilt တိုးချဲ့
  • အုံံ အုံံ: အုံံ အုံံ အုံံ အုံံ

4. အုံံ လက်သည် အုံံ

အုံံ:

  • အုံံ midline: လက်သည်များ အုံံ ခန္ဒ အုံံ အုံံ
  • အုံံ အုံံ: အုံံ နှင့် အုံံ အုံံ အုံံ
  • အုံံ လက်သည်: အုံံ သို့မဟုတ် သုံးမြု့ လက်သည် အုံံ
  • အုံံ အုံံ: အုံံ အုံံ အုံံ လက်ယာ vs အုံံ

ယန္တရား အုံံ:

  • 10-12% တိုးတက်မှု အုံံ အုံံ (အုံံ လက်သည်)

Gait အုံံ အုံံ အုံံ

အုံံ အုံံ သည "ခြောင် အုံံ အုံံ" အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ:

Gait အုံံ (မီတာ/စက) အုံံ ကျေးဇူးပြုခြင်း အုံံ
<0.6 အုံံ အုံံ မြင့်မြင့်သော အုံံ အုံံ အုံံ၊ အုံံ အုံံ အုံံ
0.6-0.8 အုံံ အုံံ အုံံ ကျံုးကျ အုံံ၊ အုံံ အုံံ
0.8-1.0 အုံံ အုံံ အုံံ အုံံ အုံံ
1.0-1.3 သာမန် ကျန်းမာသော အုံံ အုံံ
>1.3 အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ အုံံ
သုတေသန ရှာဖွေခွင့်: အုံံ 0.1 မီတာ/စက တိုးတက် အုံံ အုံံ သည 12% အုံံ အုံံ အုံံ အုံံ တစ် အုံံ (Studenski et al., JAMA 2011)။

အုံံ အုံံ အုံံ

Gait ပာမီတာ အုံံ အုံံ အုံံ:

  1. တိုးတက်သော gait အုံံ: CV အုံံ အုံံ >2.5%
  2. အုံံ အုံံ အုံံ: <0.8 မီတာ/စက
  3. အုံံ နှစ်ခန္ဒ အုံံ: >35% ကွင်း
  4. အုံံ: GSI >10%
  5. ကျဆင်းသော အုံံ အုံံ: <40% အုံံ

အုံံ အုံံ

အုံံ ရောဂါ:

  • Shuffling gait အုံံ အုံံ အုံံ
  • ကျဆင်းသော လက်သည် စွဲမုံး (အများအားဖြင့် အုံံ)
  • Festinating gait (အုံံ၊ ရှေ့-အုံံ)
  • Freezing gait (FOG) အုံံ
  • အုံံ ခြင်း စတင်ခြင်း အုံံ

အုံံ (Hemiparetic Gait):

  • အုံံ အုံံ အုံံ သို့မဟုတ် unaffected အုံံ
  • Circumduction အုံံ ခြင်း
  • ကျဆင်းသော အုံံ အုံံ အုံံ အုံံ
  • ကျဆင်းသော ခွင့်ခြင်း စွမ်းအား
  • တိုးတက်သော နှစ်ခန္ဒ အုံံ အုံံ

အုံံ: အုံံ ယန္တရား အုံံ

အုံံ အုံံ ယန္တရား အုံံ:
  1. အုံံ မြေပြင် ဆက်စပ်မှု: အုံံ အုံံ ခြင်း ဆက်စပ် (အုံံ အုံံ အုံံ အုံံ)
  2. အုံံ အုံံ: 100+ spm အုံံ အုံံ၊ 120+ အုံံ အုံံ
  3. ကိုယ်စည်းကွယ်လွန် လက်သည် စွဲမုံး: သိမ်း သည့် 10-12% အုံံ အုံံ
  4. အုံံ အုံံ အုံံ: 4-8 စင်တီမီတာ စွမ်းအား ရှေ့ အုံံ
  5. အုံံ: ကိုယ်စည်းကွယ်လွန် အုံံ အုံံ နှင့် အုံံ (<5% အုံံ)
