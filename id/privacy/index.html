Kebijakan Privasi untuk Walk Analytics

Terakhir Diperbarui: 10 Januari 2025 | Tanggal Berlaku: 10 Januari 2025

Pendahuluan

Walk Analytics ("kami" atau "aplikasi") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana aplikasi seluler kami (iOS dan Android) mengakses, menggunakan, dan melindungi data kesehatan dari perangkat Anda.

Prinsip Privasi Utama: Walk Analytics beroperasi pada arsitektur tanpa server, hanya lokal. Semua data kesehatan yang diakses dari Apple HealthKit (iOS) atau Health Connect (Android) tetap secara eksklusif berada di perangkat fisik Anda dan tidak pernah dikirimkan ke server eksternal, layanan cloud, atau pihak ketiga.

1. Akses Data Kesehatan

Walk Analytics terintegrasi dengan platform kesehatan asli perangkat Anda untuk menyediakan analisis latihan jalan kaki yang mendetail:

1.1 iOS - Integrasi Apple HealthKit

Pada perangkat iOS, Walk Analytics terintegrasi dengan Apple HealthKit untuk mengakses data latihan jalan kaki. Kami meminta akses hanya-baca (read-only) ke:

  • Sesi Latihan (Workout Sessions): Sesi latihan jalan kaki dengan waktu dan durasi
  • Jarak: Total jarak jalan kaki
  • Detak Jantung: Data detak jantung selama latihan
  • Energi Aktif: Kalori yang terbakar selama sesi jalan kaki
  • Jumlah Langkah: Analisis langkah yang diambil selama jalan kaki
  • Kecepatan Jalan: Metrik kecepatan untuk analisis laju (pace)

Kepatuhan Apple HealthKit: Walk Analytics mematuhi semua pedoman Apple HealthKit. Data kesehatan Anda diproses sepenuhnya di perangkat iOS Anda dan tidak pernah meninggalkannya. Kami tidak pernah membagikan data HealthKit dengan pihak ketiga, platform periklanan, atau broker data.

1.2 Android - Integrasi Health Connect

Tipe Data Kesehatan Izin Tujuan
Sesi Latihan READ_EXERCISE Untuk mengidentifikasi dan mengimpor sesi latihan jalan kaki dari Health Connect
Catatan Jarak READ_DISTANCE Untuk menampilkan total jarak per jalan kaki dan menghitung laju
Catatan Detak Jantung READ_HEART_RATE Untuk menampilkan grafik detak jantung, menghitung detak jantung rata-rata dan maksimum
Catatan Kecepatan READ_SPEED Untuk menghitung dan menampilkan kecepatan jalan dan zona laju Anda
Langkah READ_STEPS Untuk menghitung langkah selama sesi jalan kaki
Kalori Terbakar READ_TOTAL_CALORIES_BURNED Untuk memberikan gambaran pengeluaran energi selama jalan kaki

1.3 Bagaimana Kami Menggunakan Data Kesehatan

Semua data kesehatan digunakan secara eksklusif untuk tujuan berikut:

  • Tampilan Latihan: Menampilkan sesi jalan kaki Anda dengan metrik terperinci (jarak, waktu, langkah, detak jantung)
  • Analitik Performa: Menghitung zona laju, Skor Aktivitas, dan Kesiapan Jalan Kaki
  • Pelacakan Kemajuan: Menampilkan tren performa, rekor pribadi (personal bests), dan ringkasan latihan
  • Ekspor Data: Memungkinkan Anda mengekspor data latihan Anda ke format CSV untuk penggunaan pribadi

1.4 Penyimpanan Data

üîí JAMINAN PRIVASI KRITIS:

Semua data kesehatan tetap secara eksklusif berada di perangkat fisik Anda.

  • iOS: Data disimpan menggunakan iOS Core Data dan UserDefaults (hanya di perangkat)
  • Android: Data disimpan menggunakan Android Room Database (SQLite di perangkat)
  • TIDAK ADA data yang diunggah ke server eksternal
  • TIDAK ADA data yang dikirimkan melalui internet
  • TIDAK ADA sinkronisasi cloud atau cadangan data kesehatan
  • TIDAK ADA akses pihak ketiga ke data kesehatan Anda

Satu-satunya waktu data meninggalkan perangkat Anda adalah ketika ANDA secara eksplisit memilih untuk mengekspor latihan Anda ke format CSV dan membagikan file itu sendiri.

2. Izin yang Diperlukan

2.1 Izin iOS

  • Akses HealthKit: Akses baca ke Latihan, Jarak, Detak Jantung, Energi Aktif, Langkah, dan Kecepatan Jalan
  • Perpustakaan Foto (Opsional): Hanya jika Anda memilih untuk menyimpan ringkasan latihan sebagai gambar

Anda dapat mengelola izin HealthKit kapan saja di Pengaturan iOS ‚Üí Privasi & Keamanan ‚Üí Kesehatan ‚Üí Walk Analytics.

2.2 Izin Android

  • android.permission.health.READ_EXERCISE
  • android.permission.health.READ_DISTANCE
  • android.permission.health.READ_HEART_RATE
  • android.permission.health.READ_SPEED
  • android.permission.health.READ_STEPS
  • android.permission.health.READ_TOTAL_CALORIES_BURNED
  • Akses Internet (INTERNET): Digunakan hanya untuk menampilkan konten statis dalam aplikasi.

3. Data yang TIDAK Kami Kumpulkan

Walk Analytics TIDAK mengumpulkan nama, email, lokasi, atau analitik penggunaan.

4. Pembelian Dalam Aplikasi

Langganan dikelola oleh App Store/Google Play. Kami tidak pernah melihat detail pembayaran.

5. Retensi Data

Data tetap berada di perangkat tanpa batas waktu sampai Anda menghapusnya atau menghapus aplikasi.
