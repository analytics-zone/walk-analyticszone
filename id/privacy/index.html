Kebijakan Privasi

Terakhir diperbarui: 27 Oktober 2025

Privasi Anda Terlindungi

Walk Analytics tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi Anda. Semua pemrosesan data terjadi secara lokal di perangkat Anda.

Gambaran Umum

Walk Analytics dirancang dengan privasi Anda sebagai prioritas utama. Kebijakan Privasi ini menjelaskan komitmen kami untuk melindungi informasi pribadi dan data Anda.

Singkatnya: Kami tidak mengumpulkan data Anda karena kami tidak memerlukannya. Semua yang Anda lakukan di aplikasi tetap berada di perangkat Anda.

Informasi yang TIDAK Kami Kumpulkan

Walk Analytics tidak mengumpulkan, mengirimkan, atau menyimpan:

  • Informasi pribadi (nama, email, nomor telepon, dll.)
  • Data aktivitas berjalan (jarak, langkah, kecepatan, waktu, dll.)
  • Data kesehatan (detak jantung, kalori, dll.)
  • Data lokasi atau koordinat GPS
  • Informasi perangkat atau pengidentifikasi
  • Statistik penggunaan atau analitik
  • Laporan kerusakan atau diagnostik
  • Informasi pribadi atau sensitif lainnya

Cara Kerja Aplikasi

Semua kalkulasi dan pemrosesan data di Walk Analytics terjadi sepenuhnya di perangkat Anda:

  • Data berjalan Anda hanya disimpan di penyimpanan lokal perangkat Anda
  • Semua kalkulasi (langkah, kecepatan, irama, zona, dll.) dilakukan secara lokal
  • Tidak ada data yang pernah dikirim ke server kami atau layanan pihak ketiga
  • Tidak diperlukan koneksi internet agar aplikasi berfungsi

Penyimpanan Data

Data apa pun yang Anda masukkan atau hasilkan di aplikasi disimpan secara eksklusif di perangkat Anda menggunakan mekanisme penyimpanan lokal iOS. Data ini tetap berada di bawah kendali penuh Anda:

  • Data Anda tetap berada di perangkat Anda setiap saat
  • Kami tidak dapat mengakses data Anda
  • Jika Anda menghapus aplikasi, semua data akan dihapus secara permanen dari perangkat Anda
  • Data tidak disinkronkan ke iCloud atau layanan cloud apa pun kecuali Anda secara eksplisit mengaktifkan pencadangan aplikasi iOS

Layanan Pihak Ketiga

Walk Analytics tidak menggunakan layanan pihak ketiga apa pun, termasuk:

  • Tidak ada layanan analitik (Google Analytics, Firebase, dll.)
  • Tidak ada jaringan periklanan
  • Tidak ada layanan pelaporan kerusakan
  • Tidak ada integrasi media sosial
  • Tidak ada pemroses pembayaran (aplikasi ini gratis)

Izin

Aplikasi mungkin meminta izin iOS berikut hanya jika Anda memilih untuk menggunakan fitur tertentu:

  • HealthKit/Aplikasi Kesehatan: Hanya jika Anda memilih untuk mengimpor data latihan. Semua data tetap berada di perangkat Anda.
  • Notifikasi: Hanya jika Anda mengaktifkan pengingat latihan. Tidak ada data yang dikirim keluar dari perangkat Anda.

Semua izin bersifat opsional dan Anda dapat mencabutnya kapan saja melalui Pengaturan iOS.

Privasi Anak-Anak

Karena kami tidak mengumpulkan data apa pun, Walk Analytics aman untuk pengguna segala usia, termasuk anak-anak di bawah 13 tahun. Namun, kami merekomendasikan bimbingan orang tua untuk semua aktivitas kebugaran.

Perubahan pada Kebijakan Privasi Ini

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu untuk mencerminkan perubahan dalam aplikasi atau persyaratan hukum. Perubahan apa pun akan diposting di halaman ini dengan tanggal "Terakhir diperbarui" yang diperbarui.

Penting: Kami tidak akan pernah mengubah komitmen inti kami untuk tidak mengumpulkan data Anda.

Hubungi Kami

Jika Anda memiliki pertanyaan tentang Kebijakan Privasi ini atau aplikasi, silakan hubungi kami melalui halaman kontak kami.

Ringkasan

Walk Analytics adalah aplikasi yang mengutamakan privasi. Kami percaya bahwa data Anda milik Anda dan harus tetap bersama Anda.

  • ❌ Tidak ada pengumpulan data
  • ❌ Tidak ada pelacakan atau analitik
  • ❌ Tidak ada layanan pihak ketiga
  • ✅ 100% pemrosesan lokal
  • ✅ Kendali penuh atas data Anda
